2、一些tips：
(1)、定义圆周率:const double Pi=acos(-1.0)。

(2)、double的scanf格式：%lf;而printf格式为:%f。

(3)、在visual studio的gets_s放到PAT评测中要写成gets。输入字符数组要用gets或者scanf("%s",str)，他们默认添加'\0'作为结束符。

(4)、不要在同一个程序中使用cout和printf，有时候会出问题。

(5)、两个字符数组的相等与否不能用==来判别；
